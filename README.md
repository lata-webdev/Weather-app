# Weather App

A simple Weather App that fetches real-time weather data using an API and displays it in a user-friendly interface.

## Features
- Search for any city's weather.
- Displays temperature, humidity, and weather conditions.
- Responsive design for mobile and desktop.
- Uses OpenWeather API for live data.

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API

## How to Use
1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/weather-app.git
   ```
2. Open `index.html` in a browser.
3. Enter a city name and get the latest weather updates.

## API Setup
1. Get an API key from [OpenWeather](https://openweathermap.org/api).
2. Replace `YOUR_API_KEY` in `script.js` with your actual API key:
   ```js
   const apiKey = 'YOUR_API_KEY';
   ```
